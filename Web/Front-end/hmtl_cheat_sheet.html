p - Paragraph

Sirve para definir un bloque de texto. Basicamente crea un parrafo

Por ejemplo:

Este es un parrafo.


h1, h2, h3, h4, h5, h6 - headers

Sirven para definir encabezados o titulos. h1 es el de mayor importancia y h6 el de menos. Por ello h1 es el de mayor tamaño y h6 el de menor.

Por ejemplo:

mundo

como

Por ejemplo:

span - inline gruping

Sirve para agrupar elementos en linea.

Por ejemplo: Este texto tiene una parte azul en medio.


strong - Important

Denota importancia para su contenido y se muestra en negrita.

Por ejemplo: Es importante este texto.


i - Italic

Utilizado para cambiar el font a cursiva.

Por ejemplo: Hola me llamo beto.


u - Underlined

Subraya el texto.

Por ejemplo: Buenas noches prof octavio.


b - Bold

Hace que el texto se muestre en negrita, similar a strong, pero sin la implicación de importancia.

Por ejemplo: El videojuego esta dificil.


strike - Crossed

Este elemento tacha el texto. Indica que esta incorrecto o es irrelevante.

Por ejemplo: Texto tachado.


small - Smaller text

Se utiliza para texto más pequeño que el tamaño normal.

Por ejemplo: Hola mundo en pequeño.


em - Emphasized

Enfatiza el texto se ve en cursiva.

Por ejemplo: Me gustan los coches.

ul - Unordered List

Define una lista no ordenada, utilizada para agrupar un conjunto de elementos en una lista.

Por ejemplo:

ol - Ordered List

Define una lista ordenada. Los elementos dentro de la lista se numeran automáticamente.

Por ejemplo:

  2. lambo
  3. bugatti
